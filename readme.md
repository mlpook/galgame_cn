# galgame_cn

如标题所叙，本项目主要收集galgame汉化资源站点/频道

附带一个 `generate_list` 工具来生成markdown文档。

## 免责声明

本项目只是出于个人兴趣，收集一些入门门槛较低的资源站点/频道。

本项目不保证您通过这些站点/频道下载的资源是安全的，请您自行斟酌。

## 生成

```bash
cd generate_list &&
touch ../readme.md &&
   mkdir res &&
      cargo run -- -o ../readme.md -s ../sites.txt &&
      mv res ..
```
